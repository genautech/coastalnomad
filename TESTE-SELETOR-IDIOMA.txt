==============================================
🐛 SE NÃO FUNCIONAR
==============================================

1. Verifique se há ERROS NO CONSOLE (em vermelho)

2. Digite no console:
   console.log(typeof i18nManager);
   
   Deve retornar: "object"
   Se retornar "undefined" → Script não carregou

3. Verifique se as bandeiras aparecem no topo da página
   Se não aparecem → Problema no CSS ou HTML

4. Limpe o cache do navegador:
   Ctrl+Shift+R (Windows/Linux)
   Cmd+Shift+R (Mac)

5. Teste em modo anônimo/privado do navegador

==============================================
💻 COMANDOS DE DEBUG (COLE NO CONSOLE)
==============================================

# Ver idioma atual:
i18nManager.currentLanguage

# Forçar português:
i18nManager.setLanguage('pt-BR')

# Forçar inglês:
i18nManager.setLanguage('en')

# Ver preferência salva:
localStorage.getItem('preferred_language')

# Limpar preferência:
localStorage.removeItem('preferred_language')

# Ver quantas bandeiras foram encontradas:
document.querySelectorAll('.language-flag').length
// Deve retornar: 2
